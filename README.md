# Booki – Intégration HTML & CSS

![Logo Bookie](images/logo/Booki.png.LOGO.png)

## Description :

Booki est un site web permettant aux utilisateurs de trouver des hébergements et des activités dans la ville de leur choix.

Ce projet consiste à intégrer l’interface du site à partir d’une maquette fournie, en utilisant uniquement HTML et CSS.

---

## Objectifs :

- Intégrer fidèlement la maquette
- Respecter les contraintes techniques
- Produire un site fonctionnel sans JavaScript


---

## Installation et lancement :

1. Cloner le repository :
git clone https://github.com/Joueurdego/Bookie.git

2. Ouvrir le fichier `index.html` dans un navigateur web.


---

## Contraintes techniques
- HTML et CSS uniquement
- Aucun framework
- Aucun JavaScript
